# 📈 股票投資策略視覺化：定期定額 vs. 單筆投資

這個 Python 專案旨在透過歷史資料，視覺化比較特定股票或 ETF（以 00690 第一金工業 30 為例）在「定期定額」與「單筆投資」兩種策略下的報酬率差異，並疊加實際股價走勢，協助投資人直觀地評估不同投資策略的績效。

## ✨ 核心功能

* **定期定額 (DCA) 模擬**：以每月月底投入固定金額（預設 3,000 元），自動計算累積單位數、總成本與當下總報酬率。
* **單筆投資模擬**：透過對數報酬率計算單筆投入的累積總報酬率。
* **雙 Y 軸動態圖表**：
  * **左 Y 軸**：顯示「定期定額報酬率（紅線）」與「單筆投資報酬率（綠線）」。
  * **右 Y 軸**：顯示「歷史股價走勢（藍線）」。
* **自訂時間區間**：可依據需求動態調整圖表的 X 軸時間顯示間距與格式。

## 🛠️ 環境與套件需求

請確保你的開發環境已安裝 Python 3.x，並安裝以下必備的第三方套件：

```bash
pip install pandas numpy matplotlib
```

## 📁 資料格式要求

程式需要讀取 `.csv` 的歷史報價檔案，並放置於與程式碼相同的目錄下。檔案內容必須包含以下欄位：

| 欄位名稱 | 說明 | 格式範例 |
| :--- | :--- | :--- |
| **Date** | 交易日期 | `YYYY/MM/DD` (例如：2017/03/31) |
| **price** | 收盤價或淨值 | 數值 (例如：20.5) |

*(註：您可以從公開資訊觀測站或 Yahoo Finance 下載相關歷史數據並整理成此格式。)*

## 🚀 使用說明

1. 準備好 `00690.csv` 歷史數據檔案。
2. 若需更改每月扣款金額，可於程式碼中修改 `PMT = 3000` 的數值。
3. 若需更改分析的特定時間段，可修改 `start='2017-3-31'` 與 `end='2020-8-31'`。
4. 執行 Python 腳本：
5. 程式將會彈出一個 Matplotlib 視窗，顯示完整的投資比較折線圖。
